  ## 정밀티어모드 예시
  
  예시: 5번 사진을 S티어 [1, 2, 3, 4] 배열에서 2번과 3번 사이에 드롭

  1. 대결 순서 설정:
    - 티어의 왼쪽에 있을수록 높은 순위이다.
    - 사용자는 새로운 사진의 '최소 순위'를 고려하여 드래그&드랍한다.
    - 예를 들어 5번사진이 아무리못해도 4번사진은 이기고, 3번사진과는 좀 더 비교해보고 싶으면, 3번과 4번 사이에 5번사진을 놓는 식이다.
    
  2. 대결 진행:
    - 1단계: 3번 vs 새사진(5) → 새사진 승리 → 다음 대결
    - 2단계: 2번 vs 새사진(5) → 새사진 승리 → 다음 대결
    - 3단계: 1번 vs 새사진(5) → 1번 승리 → 1번 뒤에 배치 (2번째 위치)
  3. 최종 결과: [1, 새사진(5), 2, 3, 4]

  주요 변경사항:
  - ✅ 드롭 위치의 왼쪽 사진부터 대결 시작
  - ✅ 새사진이 이기면 더 높은 순위와 계속 대결
  - ✅ 기존사진이 이기면 그 사진 뒤에 배치
  - ✅ 진행상황 표시: 예시: S티어 2위 결정전
